# 3-variable 
## Primitive Data Type
- bool
- string
- int  int8  int16  int32  int64
- uint uint8 uint16 uint32 uint64 uintptr
- byte // alias for uint8
- rune // alias for int32 // represents a Unicode code point
- float32 float64
- complex64 complex128

## Composite Data Type
- Array
- Structs
- Slices
- Channels
- Maps
- Pointers
- Functions

## Default Value
- `false` for booleans
- `0` for integers
- `0.0` for floats
- `""` for strings
- `nil` 
  - pointers
  - functions
  - interfaces
  - slices
  - channels
  - maps
